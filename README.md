## ⚙️ How openclaw-prism Works

openclaw-prism watches your computer programs while they run to catch two main types of problems:

- **Injection scanning:** This means it looks for harmful code that tries to insert itself into your running programs. This kind of attack can make a program behave badly or steal data.
  
- **Execution guard:** It blocks suspicious or unauthorized tools from running. This stops some malware and hacking attempts before they start.

It also includes:

- **Data Loss Prevention (DLP):** Keeps sensitive data safe by watching for unauthorized copies or transfers.
- **File Integrity Monitoring:** Checks if important files on your system have been changed or tampered with.
  
This system works quietly, without bothering you, but it alerts you if it finds something wrong.
