# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in OpenClaw PRISM, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: **kyaclaw@proton.me**

You can expect:
- Acknowledgement within 48 hours
- Status update within 7 days
- Fix timeline communicated once the issue is triaged

## Scope

The following are in scope for security reports:
- Injection scanner bypass
- Invoke guard policy circumvention
- HMAC audit integrity compromise
- DLP pattern evasion
- File monitor blind spots
- Plugin hook bypass

## Out of Scope

- Vulnerabilities in OpenClaw itself (report to OpenClaw upstream)
- Vulnerabilities in Ollama or third-party models
- Denial of service against local daemons
